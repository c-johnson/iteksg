Template Installation

FTP Upload:

  • Open up your FTP manager and connect to your hosting
  • Browse to required directory (Normally public_html)
  • Upload the files inside Inspire folder.

Customization

Open / Inspire / index.html file and set the configuration options if you'd like. There's possibility to set your personal logo, & your Copyright Text, and choose to have smooth scroll or not, etc. The index.html file has a complete documentation to help you configuring your template.
